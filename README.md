# cryptocurrency_web_portfolio
암호화폐 웹 포트폴리오

## 개발환경

* front
  * HTML, CSS, JavaScript, TypeScript
  * Angular
  * Bootstrap
  * D3.js
* back
  * MySQL
  * JDBC, MyBatis
  * Spring(java)
  
## 프로젝트 계획

* 각 거래소 API 연동 및 데이터 저장 (Bithumb, Upbit)
* 회원 가입, 로그인, 관심 종목, 게시판
* 현재 시세
* 상세 보기 D3.js를 사용하여 그래프 나타 내기
* 게시판 크롤링을 통해 다른 사이트 정보 모아오기

* 기본 설계 및 준비 18.09.28 ~ 18.09.30
* 1차 완성 18.10.01 ~ 18.10.05
